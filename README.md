## ğŸš€ Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn package manager

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/Physical-Ai-Humanoid-Robotics.git
   cd Physical-Ai-Humanoid-Robotics/AI-Humanoid-Robotics/Humanoid-Book-Website
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run start
   ```

4. Open your browser to `http://localhost:3000` to view the documentation.

### Build for Production
```bash
npm run build
```
